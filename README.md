# Periodic Table of HTTP Status Codes

An interactive periodic table visualization of HTTP status codes. Browse, search, and explore all standard HTTP response codes organized by category.

**[View Live](https://periodic-table-of-status-codes.vercel.app)**

## Features

- Color-coded categories: 1xx Informational, 2xx Success, 3xx Redirection, 4xx Client Error, 5xx Server Error
- Search by code number, name, description, or symbol
- Filter by category via legend or row headers
- Detail panel with full descriptions and RFC references
- Keyboard navigation with arrow keys

## Development

It's a single `index.html` file. Open it in a browser.
